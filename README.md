# Python Automation (Phase 1)

## Overview
This repo contains Python scripts demonstrating automation concepts, including loops, conditionals, and AWS CLI integration.

## Features
- Examples of Python basics (variables, loops, functions)
- Automation scripts for AWS EC2 provisioning
- Step-by-step documentation for troubleshooting

## Setup
1. Clone the repo
2. Install Python 3.10+
3. Run scripts with `python script_name.py`

- Add Bash equivalents
